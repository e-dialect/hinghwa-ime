# 莆仙话输入方案

基于 [Rime](https://rime.im/) 的莆仙话拼音输入方案。

目前本方案适用于莆田市区口音（城里腔），其他口音使用者如果有词表也可以联系 [@Yaryou](https://github.com/Yaryou) 制作成输入码表。

## 拼音方案

### 设计理念

由于莆仙话并无清浊对立，拼音的声母采用类普拼式，方便初学者学习掌握；韵母采用国际音标转写式，更为精确。
